import re, collections

## 예제

### 단어 사전

corpus에서 주어진 `단어 : 빈도수` 사전이 다음과 같다고 가정하자.
```python
vocab = {'장 난 꾸 러 기 </w>': 5,
         '잠 꾸 러 기 </w>': 6,
         '장 난 감 </w>': 10,
         '잠 수 </w>': 3,
         '욕 심 </w>': 4}
```
초기 단어 사전의 형태는 다음과 같다.
```python
initial_vocab = set()
for word in vocab.keys():
    symbols = word.split()
    initial_vocab.update(symbols[:-1])
pprint(initial_vocab)
```
```python
{'장', '욕', '감', '난', '수', '심', '러', '잠', '꾸', '기'}
```
